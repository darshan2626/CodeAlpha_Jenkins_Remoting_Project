# CodeAlpha Jenkins Remoting Project

## Objective

To set up Jenkins Remoting by configuring a Jenkins Agent (Node) and executing a build remotely.

## Tools Used

* Ubuntu Linux
* Jenkins
* Java 21

## Steps Performed

1. Installed Java and Jenkins.
2. Configured Jenkins Dashboard.
3. Created a new Jenkins Agent (Node).
4. Connected the Agent using Jenkins Remoting.
5. Created a Freestyle Project.
6. Added shell commands for execution.
7. Ran the build successfully on the remote agent.

## Output

The Jenkins Agent connected successfully and executed the build job without errors.
