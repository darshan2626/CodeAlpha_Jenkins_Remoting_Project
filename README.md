# CodeAlpha Jenkins Remoting Project

##Objective 
Set up jenkins Remoting to connect remote jenkins nodes and distribute build execution.

##Tools Used 
- Ubuntu 26.04
- Jenkins 2.569
- OpenJDK 21 
- Jenkins Agent

## Tasks Performed
- Installed Jenkins
- Configured jenkins Dashboard
- created Agent Node
- Connected Agent using jenkins Remoting
- Executed Build on Remote Agent
- Verified Successful Build Execution 

## Output
Build executed successfully on remote Jenkins Agent.
